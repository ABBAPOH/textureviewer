## PKM Format

[Format description](https://community.arm.com/graphics/f/discussions/2248/pkm-header-format)

### Version 1.0

| Format    | Supported |
| --------- |:---------:|
| RGB8_ETC1 |    yes    |


### Version 2.0

| Format           | Supported |
| ---------------- |:---------:|
| RGB8_ETC1        |    yes    |
| RGB8_ETC2        |    yes    |
| RGBA8_ETC2_EAC   |    yes    |
| RGB8_Alpha1_ETC2 |    yes    |
| R11_EAC_UNorm    |    yes    |
| RG11_EAC_UNorm   |    yes    |
| R11_EAC_SNorm    |    yes    |
| RG11_EAC_SNorm   |    yes    |

### Writing

Writing PKM files is **not** supported
